8086run
=======

教材用のシンプルな8086エミュレータです。


特徴
----

* 1ファイル完結
* BIOSが不要
* パブリックドメイン
* Windows/POSIXに対応
* サポートする記憶装置: FDD(360KB,720KB,1440KB)×2 (HDD未対応)


起動を確認したOS
----------------

* FreeDOS (16bitカーネル)
    * 公式サイトにあるフロッピーイメージは32bitカーネルのため動きません。
* OSZ (0.0.1, 0.0.2)


使用方法
--------

```
$ ./8086run disk.img
```
