8086run
=======

教材用のシンプルな8086エミュレータです。


特徴
----

* 1ファイル完結
* BIOSが不要
* パブリックドメイン
* Windows/POSIXに対応
* サポートする記憶装置は1.44MB FDD×1のみ（HDD未対応）


起動を確認したOS
----------------

* FreeDOS 0.82pl2/3
* OSZ 0.0.1.alpha


使用方法
--------

```
$ ./8086run disk.img
```
